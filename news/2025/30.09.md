## SVG + CSS components for Svelte {#svg-css-svelte}

Following a set of Vue components, new easy to use icon components for Svelte have been published.

See [SVG + CSS for Svelte](/docs/usage/svg-css/svelte/index.md) for details.

Components use Svelte runes, Svelte version 5 is required.

There are over 200 icon component packages, one per icon set:

- `[npm]@iconify-vue/material-symbols` for Material Symbols icons
- `[npm]@iconify-vue/solar` for Solar icons
- `[npm]@iconify-vue/tabler` for Tabler icons

... and so on.

To import an icon component, import default item from `[npm]@iconify-svelte/prefix/icon` where "prefix" is icon set prefix, "icon" is icon name.

Example:

```js
import GithubIcon from "@iconify-svelte/ri/github-line";
```

then in Svelte template use:

```xml
<GithubIcon height="1em" />
```

These packages will be automatically updated when Iconify icon sets are updated.
