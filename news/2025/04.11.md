## Iconify for Tailwind CSS update {#tailwind-custom-sets}

[Iconify plugin for Tailwind 4](/docs/usage/css/tailwind/tailwind4/index.md) has been updated.

There are two changes:

- Plugin now supports custom icon sets.
- Fixed `[prop]square` option that was not working in dynamic selectors plugin.

See [custom icons in Tailwind 4](/docs/usage/css/tailwind/tailwind4/custom-icons.md) documentation.
