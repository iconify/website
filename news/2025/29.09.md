## SVG + CSS components for Vue {#svg-css-vue}

New easy to use icon components for Vue have been published.

See [SVG + CSS for Vue](/docs/usage/svg-css/vue/index.md) for details.

There are over 200 icon component packages, one per icon set:

- `[npm]@iconify-vue/material-symbols` for Material Symbols icons
- `[npm]@iconify-vue/solar` for Solar icons
- `[npm]@iconify-vue/tabler` for Tabler icons

... and so on.

To import an icon component, import default item from `[npm]@iconify-vue/prefix/icon` where "prefix" is icon set prefix, "icon" is icon name.

Example:

```js
import GithubIcon from "@iconify-vue/ri/github-line";
```

then in Vue template use:

```xml
<GithubIcon height="1em" />
```

These packages will be automatically updated when Iconify icon sets are updated.
