## SVG + CSS components for SolidJS {#svg-css-solid}

Following a set of Vue, Svelte and React components, new easy to use icon components for SolidJS have been published.

See [SVG + CSS for SolidJS](/docs/usage/svg-css/solid/index.md) for details.

There are over 200 icon component packages, one per icon set:

- `[npm]@iconify-solid/material-symbols` for Material Symbols icons
- `[npm]@iconify-solid/solar` for Solar icons
- `[npm]@iconify-solid/tabler` for Tabler icons

... and so on.

To import an icon component, import default item from `[npm]@iconify-solid/prefix/icon` where "prefix" is icon set prefix, "icon" is icon name.

Example:

```jsx
import GithubIcon from "@iconify-solid/ri/github-line";

function GithubLink() {
  return (
    <a href="https://github.com/iconify">
      <GithubIcon height="1em" />
    </a>
  );
}
```

These packages will be automatically updated when Iconify icon sets are updated.
